# View API docs

1) go to the `/docs` folder
2) run `npm install`
3) run `npm run view`
4) go to the http://localhost:5000 to see the docs in action

# Run API
## For the 1st time
1) go to the project root
2) execute
```
gem install bundler
bundle install
rails db:create
rails db:migrate
rails db:seed
rails s
```

## When the project is already set up
```
rails s
```
